RESUMEN DEL CAPITULO 9

Lo mas importante en este proyecto es el uso de usestate para inicializar valores y guardar nuevos, como se inicializo los efectos secundarios, como se limpio los efectos secundarios y como se defienen ganchos personalizados para agruparlos en la gestion de useState. El uso de useState en cada uno de los componentes fue de gran importancia para inicializar valores e ingresar nuevos que usaremos posteriormente para realizar operaciones matematicas. Tambien aprendimos a utilizar el atributo key para desmontar un componente y volverlo a montar, limpiar efectos y darle un nuevo estado al mismo.
